## 1. 사전 준비

### 1-1. Gemini API 키 발급

1. [Google AI Studio](https://aistudio.google.com/apikey) 접속
2. Google 계정으로 로그인

- **Python 3.10 이상** 필요
- 가상환경 사용 권장

## 3. 실행 방법

### 3-1. RFP 분석만 하기 (제안서 생성 없음)

RFP PDF/DOCX를 분석해 프로젝트명, 발주처, 요구사항, 평가 기준 등을 확인할 때:

```bash
python main.py analyze input/rfp.pdf
```

- `input/rfp.pdf` 대신 실제 RFP 파일 경로를 넣으면 됩니다.
- 분석 결과가 터미널에 출력됩니다.

### 3-2. 제안서 생성 (PDF 한 개 기준)

RFP 한 개로 제안서 PPTX를 만들 때:

```bash
python main.py generate input/rfp.pdf -n "프로젝트명" -c "발주처명"
```

**옵션:**

| 옵션 | 설명 | 예시 |
|------|------|------|
| `-n` / `--name` | 프로젝트명 | `-n "2026년 SNS 운영"` |
| `-c` / `--client` | 발주처명 | `-c "[발주처명]"` |
| `-t` / `--type` | 제안서 유형 | `-t marketing_pr` |
| `-o` / `--output` | 출력 폴더 | `-o output` |
| `--save-json` | 콘텐츠 JSON 저장 | 플래그만 추가 |

**제안서 유형 (`-t`):**

- `marketing_pr` — 마케팅/PR/소셜미디어
- `event` — 이벤트/행사
- `it_system` — IT/시스템
- `public` — 공공/입찰
- `consulting` — 컨설팅
- `general` — 일반 (기본)

**예시 (마케팅/PR, 유형 지정):**

```bash
python main.py generate input/rfp.pdf -n "2026년 SNS 채널 운영" -c "[발주처명]" -t marketing_pr
```

**예시 (결과물 + JSON 저장):**

```bash
python main.py generate input/rfp.pdf -n "프로젝트명" -c "발주처" --save-json
```

생성된 PPTX는 기본적으로 `output/프로젝트명_제안서.pptx` 에 저장됩니다.

## 4. 실행 흐름 (generate 시)

1. **Phase 1: 콘텐츠 생성 (Gemini)**  
   - RFP 파싱 → Gemini로 분석 → Phase 0~7 콘텐츠 생성  
2. **Phase 2: PPTX 생성**  
   - 생성된 콘텐츠를 Modern 스타일 PPTX로 변환  
3. **완료**  
   - `output/프로젝트명_제안서.pptx` 생성

처음 실행 시 Gemini API 호출이 여러 번 이루어져 1~3분 정도 걸릴 수 있습니다.

## 5. 문제 해결

### "GEMINI_API_KEY가 설정되지 않았습니다"

- 프로젝트 루트에 `.env` 파일이 있는지 확인
- `.env` 안에 `GEMINI_API_KEY=실제키` 형태로 한 줄 입력
- 또는 터미널에서 `echo %GEMINI_API_KEY%` (Windows), `echo $GEMINI_API_KEY` (macOS/Linux)로 값이 나오는지 확인

### "지원하지 않는 파일 형식"

- `generate` / `analyze` 인자로 넘기는 파일이 **PDF** 또는 **DOCX** 인지 확인

### input 폴더에 RFP가 없음

- `input` 폴더를 만들고 RFP PDF/DOCX를 넣은 뒤, 해당 경로를 인자로 지정  
  예: `python main.py generate input/제안요청서.pdf -n "프로젝트" -c "발주처"`

### 회사 정보 반영

- `company_data/company_profile.json` 에 회사 정보를 두면 제안서에 반영됩니다.  
- `generate` 시 `-d company_data/company_profile.json` 처럼 `--company` 옵션으로 경로 지정 가능 (기본값이 이미 이 경로임).

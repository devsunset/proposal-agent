# 제안서 에이전트 소개

RFP(제안요청서) 문서를 입력받아 **Impact-8 구조의 PPTX 제안서**를 자동 생성하는 AI 에이전트입니다.

---

## 핵심 가치

| 항목           | 설명 |
| -------------- | ---- |
| **표준화된 구조** | Impact-8 Framework로 검증된 제안서 구조 자동 적용 |
| **품질 평준화**   | 담당자 경험과 무관하게 일정 품질 보장 |
| **노하우 축적**   | 우수 제안서 패턴이 프롬프트·시스템에 반영 |

**Before** → **After**: 제안서 품질이 담당자 역량에 의존하던 구조에서, 검증된 구조와 LLM이 자동 적용되어 누구나 기준 이상의 제안서를 생성할 수 있습니다.

## 빠른 시작

1. [2.INSTALL_AND_USAGE.md](2.INSTALL_AND_USAGE.md) 1~2절에 따라 환경 구성 및 `.env` 설정
2. `python main.py setup-company` (최초 1회)
3. RFP 파일을 `input/`에 넣고 `python main.py generate input/파일.pdf -n "프로젝트명" -c "발주처"` 실행 (프로젝트명·발주처 미입력 시 RFP에서 자동 추출)
4. `output/`에서 생성된 PPTX 확인. 중단된 실행은 `python main.py generate --resume-checkpoint run_YYYYMMDD_HHMMSS` 로 이어서 진행 가능 (상세: [2.INSTALL_AND_USAGE.md](2.INSTALL_AND_USAGE.md) § 체크포인트 재개)

**수동 모드**(LLM API 없이 사람이 Gemini 등에 붙여넣기): `python main.py generate input/파일.pdf --manual` 후 `manual_req_res/N_step_{Phase명}_request.txt` → 응답을 `manual_req_res/N_step_{Phase명}_response.txt`에 저장 → `python main.py continue` 반복. 상세는 [2.INSTALL_AND_USAGE.md](2.INSTALL_AND_USAGE.md) 참고.

상세 단계와 옵션은 [2.INSTALL_AND_USAGE.md](2.INSTALL_AND_USAGE.md)를 참고하세요.
